# node-typescript-boilerplate

A simple node typescript boilerplate

## How To Use

Just clone the repo and start your project development.

## Available Scripts
- start:prod
  - build and run
- start:dev
  - run with nodeman and ts-node
- build
  - hmm... just build
- lint
  - let eslint help you maintain your code
